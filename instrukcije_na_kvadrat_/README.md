# Instrukcije na kvadrat

Statična web stranica za uslugu instrukcija (osnovna, srednja, fakultet).

## Stranice
- index.html (Početna)
- news.html (Vijesti)
- gallery.html (Galerija)
- about.html (O nama) — uključuje YouTube video
- contact.html (Kontakt) — uključuje Google kartu i kontakt formu

## Pokretanje
Otvorite `index.html` u pregledniku.
